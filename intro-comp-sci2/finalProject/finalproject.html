Chicago Crime Stats

by Mark De Paula and Winston Peng

About Our Site

Our website displays crime statistics from the Chicago Police Database mostly from 2015. The data is sorted by ID rather than date, so a few data entries are not part of the 2015 range. Our website displays the data in a table and gives the user a form from which they can pick a specific crime. The python file will then create a new table containing only data which contains that crime, or only crimes of which an arrest was made or was a domestic issue.
Final Project Website: Chicago Crime Stats

How To Use Site

To use the site, all you need to do is scroll and click the form. To view the full table (it is very long), you must scroll down the page. To change the table you are viewing, you click on the drop-down-table form to select the crime and then the "check data" submit button to change the table.
